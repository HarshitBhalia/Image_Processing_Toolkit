# Image Processing Toolkit

An advanced Image Processing Toolkit built using Flask, OpenCV, NumPy, and Scikit-Image that provides a wide range of digital image processing operations through an interactive web interface.

## Features

### Image Transformations

* Scaling
* Rotation
* Translation
* Shearing
* Reflection
* Image shifting

### Intensity Transformations

* Negative Transformation
* Log Transformation
* Inverse Log Transformation
* Gamma Correction
* Contrast Stretching
* Piecewise Linear Transformation
* Histogram Equalization

### Image Filtering

* Box Filter
* Gaussian Filter
* Median Filter
* Min/Max Filter
* Midpoint Filter
* Arithmetic Mean Filter
* Geometric Mean Filter
* Harmonic Mean Filter
* Contra-Harmonic Mean Filter
* Alpha-Trimmed Mean Filter

### Edge Detection & Sharpening

* Sobel Operator
* Prewitt Operator
* Laplacian Operator
* Laplacian of Gaussian (LoG)
* Canny Edge Detection
* Unsharp Masking
* High Boost Filtering

### Feature Detection

* Harris Corner Detection
* Hough Line Detection
* Hough Circle Detection
* SIFT Feature Detection
* Histogram of Oriented Gradients (HOG)
* Local Binary Pattern (LBP)

### Morphological Operations

* Dilation
* Erosion
* Opening
* Closing
* Hit-or-Miss Transform
* Boundary Extraction
* Skeletonization

### Color Processing

* BGR to Grayscale Conversion
* BGR to CMY
* BGR to CMYK
* BGR to YCbCr
* Intensity Slicing
* Pseudo Coloring

### Compression Techniques

* Run Length Encoding (RLE)
* Huffman Encoding Visualization
* DCT-based Compression
* Zig-Zag DCT Compression
* Uniform Quantization
* K-Means Color Compression

### Bitwise Operations

* AND
* OR
* XOR
* NOT

## Tech Stack

* Python
* Flask
* OpenCV
* NumPy
* Scikit-Image
* HTML/CSS/JavaScript

## Project Highlights

* Built a modular backend architecture for applying multiple image processing algorithms dynamically.
* Implemented both spatial domain and frequency domain image processing techniques.
* Designed an interactive web-based interface for real-time image upload and processing.
* Integrated classical computer vision techniques with feature extraction algorithms such as HOG, LBP, and SIFT.

## Objective

The objective of this project is to provide a comprehensive learning and experimentation platform for Digital Image Processing concepts, computer vision techniques, and image compression algorithms using Python and OpenCV.
